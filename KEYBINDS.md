# default keybinds for elijw's dwm

Feel free to modify these to your liking in [config.def.h](config.def.h).

## Master sheet

| key-combo | action |
| --- | --- |
| `SUPER` + `Return` | Spawn a terminal. (default = kitty) |
| `SUPER` + `Q` | Kill current X client. |
| `SUPER` + `SHIFT` + `Q` | Kill dwm session. (maybe change this lol) |
| `SUPER` + `B` | Toggle status bar. |
| `SUPER` + `SHIFT` + `SPACE`| Toggle floating (snap client back into current layout) |
| `SUPER` + `J` | Cycle focused client (+). |
| `SUPER` + `SHIFT` + `J` | Move focused client (+). |
| `SUPER` + `K` | Cycle focused client (-). |
| `SUPER` + `SHIFT` + `K` | Move focused client (-). |
| `SUPER` + `H` | Move master gap (L). |
| `SUPER` + `L` | Move master gap (R). |
| `SUPER` + `Z` | "Zoom" |
| `SUPER` + `I` | incnmaster (+1) |
| `SUPER` + `D` | incnmaster (-1) |
| `SUPER` + `ALT` + `0` | Toggle vanity gaps. |
| `SUPER` + `CTRL` + `,` | Cycle layouts array (-). |
| `SUPER` + `CTRL` + `.` | Cycle layouts array (+). |
